Contributeurs : Corinne Fagno, Alexandre Galdéano, Gael Ferjani et Laura Prémillieu

L'application :
Lorsqu’une catastrophe majeure se produit, il faut pouvoir y faire face.
Nous avons mis en place une application qui permet de visualiser des feux sur une carte.
L’objectif est d’être capable de planifier l’extinction de ces feux par des robots.
La répartition de l’extinction des feux doit se faire de manière automatique.
Les robots doivent pouvoir s’adapter et aller éteindre les feux qui apparaissent au fur et à mesure de la simulation. 

Prise en main :
- mettre à jour Maven

Structure générale :
src/main/java		Dossier contenant les classes java de l'application
src/test/java		Dossier contenant les tests unitaire de l'application
src/main/resources 	Dossier contenant les ressources de l'application
data				Dossier contenant les fichiers XML des graphes de l'application
logs				Dossier contenant le fichier .log qui enregistre tout les évènements de l'application
pom.xml				fichier servant à l'import des librairies
